Assignments for April 4

PDFs: Find the Peace Corps rankings for large colleges and universities for 2010-2013 (4 years), and convert the PDF tables into a single spreadsheet with the year, college, number and rank. Using either Excel or SQLite (save the excel file as a CSV and import), answer the following question:

  • Which colleges have stayed in the top 25 all four years?

Veterans: Using this spreadsheet of living veterans by state, copy the totals from 2010 (the default view) and also the male and female breakdowns (using the gender filter) for the following age ranges: under-20, 20-24, 25-29, 30-34, along with the grand total, into a new spreadsheet. Then calculate the percent of the grand total that each category (total, male, female) represents. Which states do you see as unusual or possibly newsworthy?

Readings: USA Today on demographic changes in the past 20 years, Elliot Jaspin on using Census data to uncover racial cleansing in the United States.
