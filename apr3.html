Assignments for April 10

Version Control: Using your Github account and datareporting repository, add your Story Memo data log and any related files (data, PDFs, etc.), commit them and push them to Github so that they are visible on your account. If you make any changes to the data log, do them in the repository and push them as well. If you need a refresher, try this.

Readings: Jacob Harris on How the data sausage gets made, Getting Data from the Web.

Story Memo: Keep filling out your log and send me the Github link to it. You should be starting your analysis of data at this point.
