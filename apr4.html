Assignments for April 11

Census: Using American Factfinder, find 2011 ACS 5-year estimates for median age at first marriage among 15-to-54 year olds, by state. You'll need to add the topic (Marital Status), sex (People: Age & Sex) and State (All states within the United States) to your search selections. Download a CSV version of the result (or copy and paste the table), making sure to check the box marked "Include descriptive data element names", and then clean up the CSV so that one header row remains. Then calculate the difference between the male and female median ages for each state - which state has the largest gap? What stories might you be able to report based on this data? Save the file as an excel file and email it to me, along with your story idea(s).

Readings: Jacob Harris on How the data sausage gets made, Getting Data from the Web.

Story Memo: Keep filling out your log, and email it to me (or send me a link to it if online).
