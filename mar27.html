Assignments for April 3
p>Census: Using American Factfinder, find 2011 ACS 5-year estimates for median age at first marriage among 15-to-54 year olds, by state. You'll need to add the topic (Marital Status), sex (People: Age & Sex) and State (All states within the United States) to your search selections. Download a CSV version of the result (or copy and paste the table), making sure to check the box marked "Include descriptive data element names", and then clean up the CSV so that one header row remains and save it as an Excel file. Then calculate the difference between the male and female median ages for each state - which state has the largest gap? What stories might you be able to report based on this data? Save the file as an excel file and email it to me, along with your story idea(s).

Critique: Please read this Dallas Morning News piece on college tuition and its online elements and write at least 600 words on your assessment of it. Like the first critique, imagine that you had been assigned to do this story: what would you have done differently, or not at all? Remember to be specific.
