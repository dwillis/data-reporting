Midterm (due by March 20)

Excel: Download this CSV of Census data from U.S. congressional districts and open it in Excel. Each of you has been assigned a state to focus on. Using a filter, isolate the rows for your state, and then do the following:

  1. Create a column to calculate each district's percentage of the total population that the 20-24 year-old age range represents. Which district has the highest percentage of 20-24 year olds?
  2. Do the same thing for Veteran Status:Civilian Veterans. Which district has the lowest percentage of civilian veterans?
  3. Do the same for Male & Female. Which district has the largest percentage gap between men and women?
  4. Create a row for your state and calculate the average number of people with a bachelor's degree for the state as a whole, and then add a column to calculate the difference between each district and the average.

SQL: Continuing with the senate voting database, write SQL to answer the following questions. You will need to study the fields in your tables.

  1. How many votes on cloture motions were agreed to?
  2. What is the average number of Republican No votes on a vote where the question is On Passage of the Bill?
  3. Which Democratic senator has agreed with his/her party's majority position the most times?

Email me the excel file with your "answers" in bold and the SQL from your queries by 5:20 p.m. on March 20.

Assignments for March 20

PDFs: Find the Peace Corps rankings for large colleges and universities for 2010-2013 (4 years), and convert the PDF tables into a single spreadsheet with the year, college, number and rank. Using either Excel or SQLite (save the excel file as a CSV and import), answer the following question:

  • Which colleges have stayed in the top 25 all four years?
